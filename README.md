## 🚀 Mise à jour du déploiement

### Remplacer index.html
- Aller sur le repo GitHub
- Cliquer sur `index.html` → icône crayon ✏️ → **Upload file**
- Ou glisser-déposer le nouveau `index.html` directement
- Commit : `Update — Quiz Révision Bac v21`

### Vérifier que GitHub Pages est actif
- Settings → Pages → Branch : `main` / `(root)` → **Save**
- URL : `https://nicayjam.github.io/[nom-du-repo]/`
